About The Project

MobyGlobal is a state-of-the-art whale detection network that uses acoustic monitoring and artificial intelligence to track whale movements across the world's oceans. Our system helps marine biologists study migration patterns, assists in conservation efforts, and helps prevent ship strikes.

The network consists of dozens of underwater sensors that capture audio data which is then analyzed by our custom neural network to detect whale vocalizations with over 95% accuracy.
